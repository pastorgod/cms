# el的css变量

打开F12查看元素-style-变量
:root 全局作用域下的，可以通过common.css覆盖

el的变量 如： --el-button-size: 40px;
因为渲染的时候会使用 库定义重新覆盖，所以在common.css中定义 优先级就低，
我们自己给 el-button加的样式，优先级更高一点，
所以在 我们自己写的 类似 login-btn样式中，覆盖定义的 el-button-size变量

```css
 .login-btn{
    margin-top: 10px;
    width: 100%;
    --el-button-size:50px;
  }
```
